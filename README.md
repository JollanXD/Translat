# AniList Russian

**AniList Russian** — это скрипт для [Tampermonkey](https://www.tampermonkey.net/), который переводит сайт AniList на русский язык.

## Что делает скрипт

- Переводит названия аниме и манги.
- Переводит описания, персонажей и сотрудников.
- Переводит даты, сезоны, количество серий/глав, рейтинги и прочую информацию.
- Поддерживает форматирование текста (курсив, жирный, подчеркивание, BB-коды).
- Получает русские названия и описания с сайта Shikimori, если их нет на AniList.
- Кэширует переводы, чтобы сайт работал быстрее.
- Автоматически обновляет словарь раз в день.

## Как пользоваться

1. Установите [Tampermonkey](https://www.tampermonkey.net/) в браузер
2. После установки в настройках расширений браузера в расширении менеджера скриптов прим: tampermonkey включите ***Разрешить пользовательские скрипты***
3. Перейдите по ссылке на скрипт: [УСТАНОВИТЬ](https://jollanxd.github.io/Translat/Anilist_Russian.user.js)
4. Нажмите **УСТАНОВИТЬ** в Tampermonkey.
5. Перейдите на AniList и наслаждайтесь переводом!

## Особенности

- Оригинальные описания AniList остаются доступными через раскрывающиеся блоки.
- Скрипт работает на всех страницах AniList.
- Не собирает личные данные пользователя.

Перевод интерфейса сайта AniList на русский язык.

Скрипт автоматически переводит элементы интерфейса (меню, кнопки, статусы и т.д.) с помощью локального словаря. Работает без перезагрузки страниц и поддерживает динамические обновления сайта.

🔹 Возможности:
- Перевод интерфейса (Home, Trending, Search и др.)
- Поддержка SPA (работает при переходах без перезагрузки)
- Быстрая работа
- Использование собственного словаря (GitHub)

🔹 Особенности:
- Не переводит всё 100% (ограничение структуры сайта)
- Перевод основан на словаре и может дополняться

🔹 Установка:
Просто установите скрипт через Tampermonkey и обновите страницу AniList. [install](https://jollanxd.github.io/Translat/docc/Anilist_Russian.user.js)
